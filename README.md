## About the project
Project contains simple memory game with easy and hard difficulity selection.

## Built with
* C#
* .NET Framework 4.7.2

## Installation
1. Download and unpack repository
2. Run .exe from MemoGame\bin\Release
3. Enjoy!

## Game content
After runing .exe file player needs to select difficulity level:
* Easy - 4 words to match and 10 chances
* Hard - 8 words to match and 15 chances

Player wins if all words will be matched. Otherwise if chances counter will be 0, game is over. 

Note:
For expanding words base simply edit Words.txt file in MemoGame\bin\Release
